# 🔐 Security Algorithms

A Python-based application that demonstrates and implements classical cryptography algorithms through an easy-to-use interface. This project is designed for educational purposes to help students understand encryption and decryption techniques.

## 📌 Features

- Playfair Cipher
- Polyalphabetic (Vigenère) Cipher
- Rail Fence Cipher
- Transposition Cipher
- Simplified DES (S-DES)
- User-friendly graphical interface
- Encryption and Decryption support

## 📂 Project Structure

```
security/
│── cipher_app.py                 # Main application
│── playfair_cipher.py            # Playfair Cipher implementation
│── polyalphabetic_cipher.py      # Vigenère Cipher implementation
│── rail_fence_cipher.py          # Rail Fence Cipher implementation
│── transposition_cipher.py       # Transposition Cipher implementation
│── sdes_core.py                  # Simplified DES implementation
```

## 🚀 Getting Started

### Prerequisites

- Python 3.10 or later

### Installation

1. Clone the repository

```bash
git clone https://github.com/Basmalamamdouh21/security-algorithms.git
```

2. Navigate to the project folder

```bash
cd security-algorithms/security
```

3. Run the application

```bash
python cipher_app.py
```
